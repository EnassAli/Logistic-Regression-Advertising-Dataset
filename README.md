# Logistic-Regression-Advertising-Dataset

## 📌 Project Overview
This project applies **Logistic Regression** to predict whether a user will click on an online advertisement based on their demographic and behavioral data.

The dataset includes features such as age, income, time spent online, and internet usage.

---

## 📊 Dataset
The dataset used is `advertising.csv`, which contains the following key features:

- Daily Time Spent on Site  
- Age  
- Area Income  
- Daily Internet Usage  
- Male  
- Clicked on Ad (Target Variable)

---

## ⚙️ Tasks Performed
In this assignment, I completed the following steps:

1. Loaded and explored the dataset  
2. Performed basic data analysis using `.info()` and `.describe()`  
3. Visualized the data using:
   - Histograms  
   - Jointplots  
   - Pairplots  
4. Selected relevant features for model training  
5. Split the data into training and testing sets  
6. Built a Logistic Regression model  
7. Made predictions on test data  
8. Evaluated the model using a classification report  

---

## 🧠 Model
The model used:
- Logistic Regression (from `sklearn`)

Target variable:
- `Clicked on Ad`

---

## 📈 Results
The model was evaluated using:
- Precision  
- Recall  
- F1-score  

The results show how well the model predicts user behavior in clicking advertisements.
